# kacolle-simple-helper

舰队collection简易自动化脚本

## 功能特性

- 自动点击出击中的阵型、罗盘、可选分歧点，跳过结算、掉落界面
- 添加、查看、修改不同地图的阵型、夜战、撤退配置
- 配置基地航空队（待添加）
- **不支持自动反复出击，只支持点击*出击开始*后完成当次出击**

## 使用方法

1. 安装python3.10+，并安装依赖库
    ```bash
    pip install -r requirements.txt
    ```
2. 调整游戏窗口大小为 1200x720
3. 运行脚本
    ```bash
    python main.py
    ```
